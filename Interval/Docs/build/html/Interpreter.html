visit_BinOpNode(node)¶

Visits nodes that have binary operations on them. If the token is + then we simply add all lower limit numbers and upper limit numbers separately. If the token is * then we join lower limit numbers and upper limit numbers so we can perform interval multiplication on them. :param node: :return:
